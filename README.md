# Modern Ionic & Angular Learning Repository

This repository contains my learning projects and code while following the course **"Building Modern Angular 17 Apps with Ionic and Standalone Components"** by **Simon Grimm** on YouTube.

## 📚 Course Information

- **Instructor**: Simon Grimm
- **Course**: Building Modern Angular 17 Apps with Ionic and Standalone Components
- **Platform**: YouTube
- **Original Course Version**: Angular 17 + Ionic

## 🚀 Technology Stack

This project uses modern versions of the frameworks:

- **Angular**: 20.0.0 (upgraded from Angular 17 in the course)
- **Ionic**: 8.0.0 (latest version)
- **Capacitor**: 8.3.0 (for native mobile capabilities)
- **TypeScript**: 5.9.0
- **Node.js**: Compatible with Angular 20

## 📱 What This Project Covers

This repository demonstrates the implementation of concepts from Simon Grimm's course, adapted for the latest versions of Angular and Ionic. Topics include:

- Standalone Components (Angular 17+ feature)
- Modern Ionic Framework patterns
- Mobile-first development with Capacitor
- Component architecture and best practices
- State management and data flow
- Navigation and routing in Ionic

## 🛠️ Getting Started

### Prerequisites
- Node.js (v18 or higher)
- Angular CLI
- Ionic CLI

### Installation
```bash
# Clone this repository
git clone <repository-url>
cd modern-app

# Install dependencies
npm install

# Run the development server
npm start
```

### Available Scripts
- `npm start` - Start development server
- `npm run build` - Build for production
- `npm run test` - Run tests
- `npm run lint` - Run linting
